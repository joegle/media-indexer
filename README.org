* Set up
  1. Mount directories to search in mount/
     : cd mount/
     : ln -s /Users/joe/images/ .
  2. Install and start Redis on localhost
     : redis-server 
  3. Install node packages
     : npm install
  4. Create SSL certs 
     : make ssl
  5. Start  and visit https://localhost:3000/
     : ./bin/www
